Summary

MiniMax has released M2.1, an incremental but meaningful improvement over M2, now available in Kilo Code. The update focuses on improved long-horizon reasoning and efficiency: the Kilo team reports that M2.1 performs better on tasks that require multi-step planning and shows notable gains in speed and resource usage. The release continues MiniMax’s open-source approach—model weights are shared on Hugging Face—making it easy for developers to experiment and deploy the model.

M2.1 is immediately available to Kilo Code users. The Kilo team encourages developers to stress-test the model across real-world coding tasks (code generation, debugging, and documentation), and highlights that Kilo’s Code Reviews feature can integrate M2.1 to provide AI-driven pull request feedback.

Overall, M2.1 appears to be a more focused and efficient iteration of M2: improvements are pragmatic (better reasoning on longer contexts and faster inference), and the release underlines MiniMax’s intent to remain competitive in open-source model development.
